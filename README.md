# Payment Processing System

A console-based Payment Processing System built in Python as part of a bootcamp assignment. The application simulates basic payment operations and helps understand object-oriented programming concepts.

## Features
- Add cash to the system
- Refund money
- Check account balance
- View transaction history
- Menu-driven console interface

## Technologies Used
- Python
- Object-Oriented Programming (OOP)

## Project Structure
```
payment-processing-system/
│
├── main.py
├── problem_statement.txt
└── payment_system/
    ├── __init__.py
    ├── payment.py
    └── payment_method.py
```

## How to Run
Run the program using:
```
python main.py
```

## Learning Outcome
This project helped in understanding:
- Python classes and objects
- Encapsulation
- Modular code structure
- Transaction handling logic

⭐ Beginner-friendly project to practice Python OOP concepts.
